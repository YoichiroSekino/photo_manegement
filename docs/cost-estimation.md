#### ROI計算

```
初期投資: ¥27,974,246

年間削減効果（5プロジェクト想定）:
  ¥1,100,000 × 5 = ¥5,500,000/年

ROI = (年間削減効果 - 年間運用コスト) / 初期投資
    = (¥5,500,000 - ¥11,950,320) / ¥27,974,246
    = -23%  (2年目以降プラスに転換)

投資回収期間: 約3.5年
```
